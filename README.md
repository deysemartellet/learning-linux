# 🛡️ Linux SOC Playbook
#### Repositório dedicado ao estudo prático de comandos Linux utilizados em ambientes de SOC (Security Operations Center) e atividades de Blue Team.
#### O objetivo deste projeto é documentar, de forma clara e estruturada, comandos essenciais para investigação de incidentes, análise de logs e navegação em sistemas Linux — sempre com foco em aplicação real no dia a dia de um analista de segurança.

# 🎯 Objetivo
#### Consolidar conhecimento em Linux voltado para segurança
#### Criar um material de referência rápida para investigações
#### Demonstrar habilidades práticas para oportunidades na área de cibersegurança
#### Evoluir de forma consistente rumo a um nível profissional (SOC Analyst)

# 🧠 O que você vai encontrar aqui
### 📁 commands/
##### Documentação de comandos organizados por contexto:
##### Análise de processos
##### Análise de arquivos
##### Logs do sistema
##### Atividade de usuários
##### Rede
### Cada comando inclui:
##### Explicação em linguagem simples
##### Contexto de uso em SOC
##### Exemplos práticos
##### Pontos de atenção durante investigações

### 📁 scenarios/
##### Simulações de investigações reais, como:
##### Tentativas de brute force
##### Processos suspeitos
##### Acessos não autorizados
### Incluindo:
##### Passo a passo da análise
##### Comandos utilizados
##### Interpretação dos resultados
##### Possíveis ações de mitigação

## 🛠️ Ambiente de Estudo
### Os testes e simulações estão sendo realizados em ambientes controlados, como:
##### JSLinux (simulação online)
##### Máquinas locais e ambientes isolados
###### ⚠️ Nenhuma informação sensível ou real é exposta neste repositório.
